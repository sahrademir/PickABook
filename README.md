📖 PickABook
AI-Powered Hybrid Book Recommendation System

PickABook is an end-to-end AI recommendation engine that suggests books using multiple recommendation techniques. The system combines classical Natural Language Processing (TF-IDF), semantic embeddings generated by Sentence Transformers, FAISS similarity search, and Large Language Models (LLMs) to provide accurate and explainable book recommendations through an interactive Streamlit interface.

✨ Features:
📚 Content-Based Recommendation (TF-IDF + Cosine Similarity)
🧠 Deep Learning Semantic Search (Sentence Transformers)
⚡ FAISS Vector Index for fast retrieval
👤 Personalized User-to-Book Recommendation
🤖 LLM-generated explanations for every recommendation
📖 Book cover visualization
🎨 Modern Streamlit interface
🔍 Interactive book search
⭐ Recommendation confidence score
🚀 End-to-end AI pipeline

## Additional Model Files

Due to GitHub file size limitations, the FAISS index and metadata files are not included in this repository.

They can be downloaded from Google Drive:

- books_index.faiss
- books_metadata.pkl

After downloading, place both files inside:

data/

before running the application.

- [Download books_index.faiss](https://drive.google.com/file/d/1rDPGhtlzkNkUA3kTCIbUInRgfV-OrMTQ/view?usp=drive_link)
